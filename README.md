# IAT Experiment


一个基于 React + Vite + Tailwind CSS 构建的 IAT（内隐联想测试）实验项目。

## 技术栈

- React 19
- Vite 7
- Tailwind CSS 4
- Lucide React Icons

## 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

## 安装

```bash
# 克隆仓库
git clone https://github.com/你的用户名/iat-experiment.git

# 进入项目目录
cd iat-experiment

# 安装依赖
npm install
```

## 开发

```bash
# 启动开发服务器
npm run dev
```

开发服务器默认运行在 http://localhost:5173

## 构建

```bash
# 构建生产版本
npm run build
```

构建产物将输出到 `dist/` 目录。

## 预览

```bash
# 预览生产构建
npm run preview
```

## 代码检查

```bash
# 运行 ESLint
npm run lint
```

## 项目结构

```
iat-experiment/
├── public/          # 静态资源
├── src/             # 源代码
├── dist/            # 构建输出
├── index.html       # 入口 HTML
├── vite.config.js   # Vite 配置
├── package.json     # 项目配置
└── README.md        # 项目说明
```
